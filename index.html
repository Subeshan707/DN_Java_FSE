Exercise 10 — Chrome DevTools

Debugging Guide

🔍 Inspect Element

Right-click any element → Inspect. Edit HTML/CSS live in the Elements panel. Try changing .eventCard padding on the fly.

🖥 Console Tab

Open DevTools → Console. All console.log() messages from this page appear here. Try: localStorage.getItem('preferredEvent')

🔴 Breakpoints

Sources tab → open script → click line number. Reload page. Execution pauses so you can inspect variable values step-by-step.

📱 Device Toolbar

Press Ctrl+Shift+M (or the phone icon) to simulate tablets and phones. Test your media queries live.

🌐 Network Tab

Reload the page with Network tab open to verify styles.css loads (Status 200). Check font and image requests too.

💾 Application Tab

See localStorage and sessionStorage values live. You'll find preferredEvent key after selecting an event above.
