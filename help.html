How to Register for an Event

  1. Go to the Home Page
  2. Click "Register for an Event" in the navigation
  3. Fill in your name, email, date, and choose an event type
  4. Click "Confirm Registration" — a confirmation message appears via <output>

Saving Preferences

Your selected event type is automatically saved in localStorage and restored on your next visit.

Finding Nearby Events

Click "Find Nearby Events" and allow browser location access. Uses the HTML5 Geolocation API.
